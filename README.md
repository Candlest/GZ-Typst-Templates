# Typst Template

## Fonts

Install the fonts by yourself.

- text: ("CMU Serif", "Source Han Serif SC"),
- sans: ("Source Han Sans SC"),
- code: ("JetBrains Mono", "Consolas"),

## Install Typst

```bash
cargo install --git https://github.com/typst/typst
```

## Usage

```bash
typst watch _usage.typ
```
